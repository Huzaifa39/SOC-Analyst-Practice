# SOC Analyst Practice Portfolio

This repository contains practical exercises for SOC L1 analyst learning using Splunk.  
It demonstrates log collection, authentication monitoring, time-based triage, and dashboard creation from Windows Event Logs in a lab environment.

## Folder Structure

- **Day1/** – Baseline discovery, authentication searches, daily triage dashboards  
- Future folders (Day2, Day3…) will contain additional SOC practice exercises

## How to Use

- Open `notes.md` in the corresponding folder to see queries, observations, and analysis.
